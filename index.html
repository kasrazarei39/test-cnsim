About

This is an event-based blockchain simulator specifically designed to simulate Bitcoin consensus network behavior. It's part of the larger CNSim (Consensus Network Simulator) framework.

Features

  • Bitcoin network behavior simulation
  • Honest and malicious node modeling
  • Transaction and block processing
  • Mining difficulty adjustment
  • Comprehensive reporting and analysis
